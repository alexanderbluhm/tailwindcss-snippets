## Tailwindcss Gradients in SVG Icons

If you want to use Tailwinds gradient utilities inside (inline) SVGs you can do that using this simple trick:

<div className="flex items-center space-x-8">
  <svg
    className="w-24 h-24 from-purple-500 to-blue-800 transform hover:scale-105 transition-all"
    viewBox="0 0 24 24"
  >
    <defs>
      <linearGradient id="gradient" x2="1" y2="1">
        <stop offset="0%" stopColor="var(--tw-gradient-from)" />
        <stop offset="100%" stopColor="var(--tw-gradient-to)" />
      </linearGradient>
    </defs>
    <rect width="24" height="24" rx="4" fill="url(#gradient)" />
  </svg>
  <svg
    className="w-24 h-24 from-green-500 to-teal-500 transform hover:scale-105 transition-all"
    viewBox="0 0 24 24"
  >
    <defs>
      <linearGradient id="gradient" x2="1" y2="1">
        <stop offset="0%" stopColor="var(--tw-gradient-from)" />
        <stop offset="100%" stopColor="var(--tw-gradient-to)" />
      </linearGradient>
    </defs>
    <circle cx="12" cy="12" r="12" fill="url(#gradient)" />
  </svg>
</div>

```html
<svg className="w-64 h-64 from-purple-500 to-blue-800" viewBox="0 0 24 24">
  <defs>
    <linearGradient id="**gradient**" x2="1" y2="1">
      <stop offset="0%" stop-color="var(--tw-gradient-from)" />
      <stop offset="100%" stop-color="var(--tw-gradient-to)" />
    </linearGradient>
  </defs>
  <rect width="24" height="24" rx="4" fill="**url(#gradient)**" />
</svg>
```

Of course you can use the same trick for [radial gradients](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/radialGradient) or use different [stop](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/stop) parameters.
