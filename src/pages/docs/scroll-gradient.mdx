## Scroll Gradient

It is often useful to have a little fade when making a scrollable list.  
This can be achieved using Tailwinds `bg-gradient-to-b` and a fade to transparent.

<div class="relative max-w-sm mx-auto">
    <div class="absolute inset-x-0 z-10 h-12 pointer-events-none bg-gradient-to-b from-white"></div>
    <div class="h-48 pt-8 space-y-4 overflow-y-scroll">
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
    </div>
</div>

```html
<div class="relative">
    <div class="**absolute inset-x-0 z-10 h-12 pointer-events-none bg-gradient-to-b from-white**"></div>
    <div class="h-32 **pt-8** space-y-4 overflow-y-scroll">
        <div class="w-full h-4 bg-gray-200 rounded-full"></div>
        <!-- ... -->
    </div>
</div>
```
