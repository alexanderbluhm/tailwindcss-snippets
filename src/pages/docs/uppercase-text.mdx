## Uppercase Text

When using uppercase text it is a good idea to increase the tracking for readability reasons.

<p class="font-bold text-indigo-500 uppercase">
    Lorem ipsum dolor
</p>

<p class="font-bold tracking-wider text-indigo-500 uppercase">
    Lorem ipsum dolor
</p>

```html
<p class="text-indigo-500 font-bold uppercase **tracking-wider**">
    Lorem ipsum dolor
</p>
```
