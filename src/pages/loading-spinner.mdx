## Loading Spinner

There is a cool way to create a loading spinner only with **TailwindCSS**.

<button className="bg-gradient-to-br from-teal-300 to-blue-500 shadow-sm text-white px-3 py-2 text-sm font-medium rounded inline-flex items-center justify-center">
  Sending
  <span className="ml-2.5 inline-flex border-t-2 border-t-gray-100 border-b-white border-b rounded-full w-5 h-5 animate-spin"></span>
</button>

```html
<span
  className="inline-block border-t-2 border-t-gray-900 border-b-gray-500 border-b-2 rounded-full w-5 h-5 animate-spin"
></span>
```

However, while this is more like a fun solution. Here are some **beautiful free** svg icons to use:

- [Heroicons](https://heroicons.com) (search for _refresh_)
- [Iconic](https://iconic.app/refresh/)
- [SystemUI Icons](https://systemuicons.com) (search for _load_)
